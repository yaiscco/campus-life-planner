# campus-life-planner
Core Functionality
Responsive Layout: Uses CSS Flexbox and Grid for a mobile-first interface that adapts to various screen sizes.

Data Validation: Implements Regex to sanitize inputs, prevent leading/trailing spaces, and detect duplicate entries.

State Management: Utilizes LocalStorage to maintain data persistence across browser sessions.

Data Portability: Features JSON export and import capabilities for backup and restoration of task lists.

Technical Specifications
Structure: Semantic HTML5 with ARIA roles for accessibility.

Styling: Custom CSS3 featuring a banner-style "Starry Night" header and flex-aligned logo positioning.

Logic: Vanilla JavaScript (ES6) for real-time search filtering and interactive UI updates.
